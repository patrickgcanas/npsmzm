## O que foi feito nessa sessão

### 1. Soft delete no reset do banco
- `app/api/admin/reset/route.js` — trocado de hard delete para `updateMany({ deletedAt: now() })`
- Adicionado campo `deletedAt DateTime?` em `SurveyResponse` e `SurveyInvite` no schema Prisma
- Migration rodada manualmente no SQL Editor do Supabase:
  ```sql
  ALTER TABLE "SurveyResponse" ADD COLUMN "deletedAt" TIMESTAMP;
  ALTER TABLE "SurveyInvite" ADD COLUMN "deletedAt" TIMESTAMP;
  ```
- Todas as queries de leitura filtram `WHERE deletedAt IS NULL`
- Para zerar o banco antes de uma apresentação:
  ```js
  fetch('/api/admin/reset', { method: 'POST' }).then(r => r.json()).then(console.log)
  // retorna { ok: true, hidden: N }
  ```

## Arquitetura de autenticação

- **Sem banco** — sessão stateless com HMAC-SHA256 (Web Crypto API, roda no Edge)
- Cookie `mzm-auth` HTTP-only, 12h de expiração
- Credenciais em env vars Vercel: `ADMIN_EMAIL`, `ADMIN_PASSWORD`, `SESSION_SECRET`
- `middleware.js` protege todas as rotas exceto `/survey/*`, `/login`, `/api/auth/*`, `/api/responses/*`, `/api/invites/*/track`

## Sistema de envio em lote (bulk email)

1. Importar planilha Excel/CSV na aba **Importar / Exportar**
   - Colunas: `Nome do Cliente`, `E-mail`, `Sigla do Cliente`, `Advisor responsável`
2. Ir para aba **Enviar** — painel mostra todos os convites pendentes
3. Clicar **Abrir e-mail** em cada linha — abre rascunho no Outlook via `mailto:`
4. Para limpar a lista e começar novo lote: botão **Limpar registro de envio**

## Banco de dados — modelos principais

```prisma
model SurveyInvite {
  id          String    @id @default(cuid())
  clientName  String
  email       String
  clientCode  String?
  advisor     String
  token       String    @unique
  sentAt      DateTime?
  viewedAt    DateTime?
  startedAt   DateTime?
  deletedAt   DateTime?   // soft delete
  response    SurveyResponse?
}

model SurveyResponse {
  id          String   @id @default(cuid())
  inviteId    String   @unique
  nps         Int
  csat        Int
  pillars     Json
  comment     String?
  journeyStage String?
  submittedAt DateTime @default(now())
  deletedAt   DateTime?   // soft delete
}
```
